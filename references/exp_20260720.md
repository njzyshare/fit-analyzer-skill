## 关键踩坑

| # | 问题 | 错误做法 | 正确做法 | 原因 |
|---|------|---------|---------|------|
| 1 | timer 含暂停 | `timer = Σ(records 时间间隔)` | `timer = Σ(各段原始 session.totalTimerTime)` | records 时间差含红绿灯等自动暂停 |
| 2 | 爬升被放大 | `ascent = Σ(GPS altitude 正向差)` | `ascent = Σ(各段原始 session.totalAscent)` | GPS 海拔误差 ±3-5m，几千点累积后数百米 |
| 3 | 佳明拒收 | `protocol=16`(原始设备) 或 `protocol=32`(官方SDK) | `protocol=2`（Encoder 默认） | 原因不明，实测 protocol=2 可上传 |
| 4 | 佳明重算海拔 | 缺少 device_info | 注入 manufacturer=garmin product=4536 | 佳明对非本家设备重算海拔 |
| 5 | 配速显示 ? | session 缺 avgSpeed/avgHeartRate | 显式写入所有 session 字段 | Encoder 不会自动填充派生字段 |
| 6 | 触地时间缺失 | record 只写 core 字段 | 写入全部已知字段(verticalOscillation等) | 佳明从 record 字段读步态数据 |

## 已确认可上传的消息组合

✅ **可上传：**
- fileId + deviceInfo + sport + fileCreator + deviceSettings + userProfile
- timeInZone + event + record(全部已知字段)+ lap(26圈+步态数据) + session(含全部已知字段) + activity

❌ **不可上传（"上传出错。请重试。"）：**
- gpsMetadata / timestampCorrelation（时序辅助消息）
- split / splitSummary（分段数据）
- unknown_* 私有消息（Encoder 不支持写入）

## 字段计算规则

| 字段 | 来源 | 备注 |
|------|------|------|
| totalTimerTime | `Σ(各段原始 session.totalTimerTime)` | 手表计时器，不含暂停 |
| totalElapsedTime | `首段首条 ~ 末段末条` | 墙钟含暂停 |
| totalDistance | records 累加 | distance 校准后 |
| avgHeartRate | 加权平均运动中 records | speed>0 或 heartRate>0 |
| totalCalories | `Σ(各段原始 session.totalCalories)` | |
| totalAscent/Descent | `Σ(各段原始 session.totalAscent)` | 不从 GPS 算 |
| avgSpeed / maxSpeed | records 中 enhancedSpeed | |
| avgPower / maxPower | records 中 power | |
| avgCadence / maxCadence | records 中 cadence | |
| 步态(垂直振幅/触地时间/步长) | records 中 verticalOscillation/stanceTime/stepLength | 各圈取平均值 |
| totalTrainingEffect | 从 seg1 session 取 | 合并后不可重算 |
